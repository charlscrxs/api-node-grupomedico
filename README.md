## Tecnologías Utilizadas 
* **Lenguaje:** JavaScript (ES6+) / Node.js
* **Framework:** Express.js
* **Base de Datos:** PostgreSQL
* **Driver:** `pg` (node-postgres)
* **Seguridad & Config:** `dotenv`, variables de entorno.
* 
* ## Funcionalidades Principales 
El sistema cuenta con endpoints RESTful organizados para gestionar:
* **Médicos:** Registro, perfiles y especialidades.
* **Pacientes:** Expedientes e información personal.
* **Citas Médicas:** Agendamiento y control de horarios.
*  **Historial Clínico:** Registro de consultas, diagnósticos y tratamientos.
* **Recetas:** Vinculación de medicamentos a consultas.
* **Admin:** Gestión de usuarios y permisos (CRUD).
